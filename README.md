# Frogy
Close windows on x11 with STYLE!

Have you evor wanted to close a window, but thaught that pressing the X was boring? then this app is for you! to close a app with style, use `frogy --eat "app name"`. but this is very good because you need to press y/n, wich is BORING! So use `--auto` to kill prosseses with speed. But what if you  went to youareaidiot.com? you can only lanch frogy so fast. then use `--kill-children` and all of those pop-ups will be shown that they are the real idiots. Now I know what you are thinking: "I will just go to the releces page, and download it thare." or "let me just `sudo apt install frogy`." but it is not that simple. well... its more simple, just doenload the install script, and run it. it will make a python venv, install dependencys, and make .desktop files.
